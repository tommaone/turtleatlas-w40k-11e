# Expert Knowledge Files

Faction-specific domain knowledge for turtleatlas-w40k-11e.

Each file covers a single faction:
- Unit categories and battlefield roles
- Weapon profiles with special rules (Psychic, Torrent, etc.)
- Squad weapon limits per datasheet
- Detachment overview with rules and stratagems
- Gotchas and non-obvious behaviours unique to that faction

**These files are faction-specific.** The general 11e rules reference is in `resources/guardrails.md`.
